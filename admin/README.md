# 净水智服 · 管理后台

净水器上门服务平台的管理后台前端项目。**纯前端 UI 演示项目**，所有数据均为本地 mock 假数据，不依赖任何后端接口。

## 技术栈

- Vue 3 (`<script setup>` 组合式 API)
- Vite 5
- Element Plus（UI 组件库 + 图标 + 中文语言包）
- Vue Router 4
- Pinia（状态管理）
- ECharts 5（数据可视化 / 数据大屏）

## 快速开始

```bash
# 1. 进入项目目录
cd admin

# 2. 安装依赖
npm install

# 3. 启动开发服务器（默认端口 8000，会自动打开浏览器）
npm run dev
```

打开浏览器访问 http://localhost:8000

> 登录页：任意账号密码均可登录（mock 校验）。例如 `admin / 123456`。

## 构建

```bash
npm run build      # 打包到 dist/
npm run preview    # 本地预览打包结果
```

## 功能模块

| 模块 | 说明 |
| --- | --- |
| 登录 | 品牌渐变插画 + 账号密码表单，登录态存 Pinia + localStorage |
| 工作台首页 | 核心指标卡、营收趋势、工单状态环形图、待办、最新工单 |
| 数据大屏 | 深色炫酷全屏大屏，多 ECharts 图表 + 翻牌 + 实时滚动 |
| 数据中心 | 装机 / 充值 / 客户数据分析（图表 + 表格筛选） |
| 商城管理 | 商品列表、商品分类、订单管理（订单详情抽屉 + 物流时间线） |
| 设备管理（IoT） | 设备列表、设备详情（实时状态 + 远程控制）、设备型号管理 |
| 服务工单 | 工单列表、派单、跟单详情时间线、师傅管理 |
| 分销管理（链动 2+1） | 分销概览、分销员、佣金记录、提现审核、分润规则配置 |
| 客户管理 | 客户列表、客户详情（设备 / 订单 / 工单 tab） |
| 充值管理 | 充值套餐配置、充值记录 |
| 页面装修 | 三栏可视化装修器（组件库 / 手机预览 / 属性面板） |
| 预警管理 | 预警列表、预警规则配置 |

## 目录结构

```
admin/
├── index.html
├── vite.config.js
├── package.json
└── src/
    ├── main.js              # 入口，全量引入 Element Plus + 图标 + 中文包
    ├── App.vue
    ├── router/              # 路由
    ├── store/               # Pinia store
    ├── styles/              # 全局样式 / 主题变量
    ├── layout/              # 后台布局（侧边栏 + Header + Tabs）
    ├── components/          # 可复用组件（数据卡、图表卡等）
    ├── mock/                # 各模块 mock 数据
    └── views/               # 各业务页面
```

## 说明

- 所有列表的搜索、筛选、分页均为前端本地切片实现。
- 所有弹窗 / 抽屉 / 表单提交后仅更新前端本地状态并给出 `ElMessage` 提示。
- 图片一律使用 CSS 渐变色块 + Element Plus 图标占位，不依赖外部图片 URL。
